JDBC 3.0 Support

The SequeLink for JDBC driver supports the JDBC 3.0 API. This functionality is available only to applications running on JDK 1.4 or higher Virtual Machines.

Table 4-1 summarizes the JDBC 3.0 and enhanced JDBC 2.0 functionality that the SequeLink for JDBC driver supports.
